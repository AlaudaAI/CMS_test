# CMS Platform

## Quick Start

**Prerequisites:** Node.js 22+ and PostgreSQL (see root [README](../README.md))

```bash
bash setup.sh     # one-time: installs deps, creates DB, seeds data
npm run dev        # http://localhost:3000
```

Admin: http://localhost:3000/admin → `admin@platform.com` / `changeme123`

## Reset Everything

```bash
dropdb cms_platform
bash setup.sh
npm run dev
```

## Environment Variables

| Variable | Description |
|---|---|
| `POSTGRES_URL` | Postgres connection string |
| `PAYLOAD_SECRET` | Auth secret (auto-generated by setup.sh) |
| `BLOB_READ_WRITE_TOKEN` | Vercel Blob token (production only) |

## Deploy to Vercel

1. Connect repo → set root directory to `my-real-estate`
2. Add `POSTGRES_URL`, `PAYLOAD_SECRET`, `BLOB_READ_WRITE_TOKEN`
3. Deploy
